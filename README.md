# swift-rfc-5322

Swift implementation of [RFC 5322: Internet Message Format](https://www.rfc-editor.org/rfc/rfc5322.html)

## Installation

### Swift Package Manager

Add this to your Package.swift:

```swift
dependencies: [
    .package(url: "https://github.com/swift-web-standards/swift-rfc-5322.git", branch: "main")
]
```
